Concepto Descripción
Creación de tablas Las tablas en HTML se crean usando el elemento <table>. Son útiles para representar datos en formato de cuadrícula.
Importancia de no usarlas para maquetar Antiguamente, las tablas se usaban para maquetar páginas web. Sin embargo, esto es considerado una mala práctica hoy en día. Las hojas de estilo CSS ofrecen herramientas más flexibles y semánticas para el diseño y maquetación.
Uso de Colspan y Rowspan colspan y rowspan son atributos que permiten que una celda ocupe varias columnas o filas, respectivamente. Son útiles para combinar celdas y crear tablas más complejas.
Elementos semánticos en tablas Las tablas tienen varios elementos semánticos, incluyendo:
  • <thead>: Representa el encabezado de la tabla.
  • <tbody>: Representa el cuerpo principal de la tabla.
  • <tfoot>: Representa el pie de la tabla (aunque se define antes que <tbody> en el código, se muestra al final).
  • <tr>: Representa una fila.
  • <th>: Representa una celda de encabezado.
  • <td>: Representa una celda de datos.
